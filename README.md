## Inventory Demand Forecasting

Client: NewX Services

## Project Overview
This project focuses on forecasting spare parts inventory demand for service centers using data analysis and predictive modeling techniques.

## Business Problem
Service centers maintain large spare parts inventories which increases operational cost. Predicting demand helps optimize stock levels and improve availability.

## Project Goal
Develop a predictive model to forecast spare parts demand so service centers can implement Just-In-Time inventory management.

## Dataset
28,484 records with features such as:
- Invoice Date
- Job Card Date
- Vehicle Model
- Business Partner Name
- Current KM Reading
- Invoice Line Text
